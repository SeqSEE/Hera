### Prerequisites

- [NodeJS 12+](https://nodejs.org/en/download/)
- [TypeScript](https://www.typescriptlang.org/#installation)

### Setup

```
git clone https://github.com/SeqSEE/Hera
```

- install the dependencies

```
cd Hera
```

```
npm install
```

- build the project

```
npm run build
```

- start the bot

```
npm run start
```
